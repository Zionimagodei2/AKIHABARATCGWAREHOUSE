## Actionable SEO Recommendations

### HIGH PRIORITY

#### 1. Implement Complete Meta Tag Strategy
**Why:** Only 4/10 competitors have well-optimized meta titles. Most competitors neglect meta descriptions or make them too thin. This is a quick win.

**Actions:**
- Write unique meta titles (50-60 chars) for every page: `[Product/Category] | Akihabara TCG Warehouse`
- Write compelling meta descriptions (120-160 chars) with primary keywords + CTA
- Example Homepage: `<title>Japanese & English TCG Cards | Pokemon, One Piece, Yu-Gi-Oh | Akihabara TCG Warehouse</title>`
- Example Homepage description: `Shop authentic Japanese & English trading card games at Akihabara TCG Warehouse. Pokemon, One Piece, Yu-Gi-Oh, Weiss Schwarz, Gundam & more. Ships worldwide from Japan.`

#### 2. Add JSON-LD Structured Data
**Why:** NONE of the 10 competitors use JSON-LD structured data. This is a MASSIVE opportunity to gain rich snippets in search results.

**Actions:**
```json
// Homepage: Organization + WebSite schema
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Akihabara TCG Warehouse",
  "url": "https://akihabaratcg.com",
  "logo": "https://akihabaratcg.com/logo.png",
  "description": "Premium Japanese & English TCG card shop based in Akihabara, Tokyo"
}

// Category pages: ItemList + BreadcrumbList
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {"@type": "ListItem", "position": 1, "name": "Home", "item": "https://akihabaratcg.com"},
    {"@type": "ListItem", "position": 2, "name": "Pokemon", "item": "https://akihabaratcg.com/pokemon"},
    {"@type": "ListItem", "position": 3, "name": "Japanese Booster Boxes", "item": "https://akihabaratcg.com/pokemon/japanese-booster-boxes"}
  ]
}

// Product pages: Product schema
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Pokemon TCG Scarlet Violet Booster Box Japanese",
  "image": "...",
  "description": "...",
  "brand": {"@type": "Brand", "name": "Pokemon"},
  "offers": {"@type": "Offer", "price": "...", "priceCurrency": "JPY", "availability": "https://schema.org/InStock"}
}
```

#### 3. Fix Heading Hierarchy
**Why:** 50% of competitors use multiple H1 tags (bad practice), and some miss H1 entirely. We should be the site that does it right.

**Actions:**
- Exactly ONE H1 per page with primary keyword
- H2 for major sections (Shop by Game, New Arrivals, Best Sellers)
- H3 for subsections (Pokemon TCG, One Piece TCG, etc.)
- Example: H1: "Premium Japanese & English TCG Cards from Akihabara", H2: "Shop by Game", H3: "Pokemon TCG"

#### 4. Leverage "Akihabara" and "Japan-Sourced" as Unique Differentiators
**Why:** Only PlazaJapan and Hobby-Genki emphasize Japanese origin. Our Akihabara location is a unique SEO opportunity that no competitor can match.

**Target Keywords:**
- `japanese pokemon cards` / `japanese pokemon booster box`
- `akihabara card shop` / `akihabara tcg`
- `authentic japanese booster box`
- `japan imported tcg` / `japan import trading cards`
- `pokemon cards from japan`
- `japanese tcg cards online`
- `akihabara warehouse cards`
- `tokyo card shop online`

#### 5. Implement Complete Open Graph + Twitter Card Tags
**Why:** Most competitors have incomplete OG tags. Full social markup improves click-through from social shares.

**Actions:**

### MEDIUM PRIORITY

#### 6. Optimize Image Alt Text
**Why:** 90% of competitors use alt text, but quality varies. CardKingdom uses full scene descriptions while others use minimal text.

**Best Practice (from CardKingdom):**
- Include: Game name + Product name + Product type + Language + Condition
- Example: `alt="Pokemon TCG Scarlet Violet Booster Box Japanese Sealed"`
- NOT: `alt="image1.jpg"` or `alt="product"`

#### 7. Implement Clean URL Structure
**Why:** Top competitors use keyword-rich, hierarchical URLs. Cardmarket and CardKingdom have the best patterns.

**Recommended Structure:**
```
/pokemon                          -> Pokemon category
/pokemon/japanese                 -> Japanese Pokemon subcategory
/pokemon/japanese/booster-boxes   -> Booster boxes listing
/pokemon/japanese/booster-boxes/sv9-battle-partners  -> Product page
/one-piece/english/elite-trainer-boxes
/yu-gi-oh/japanese/structure-decks
/weiss-schwarz/english/booster-boxes
/gundam/japanese/booster-boxes
/lorcana/english/booster-boxes
```

#### 8. Add Canonical URLs
**Why:** Only 1/10 competitors use canonical tags (and PlazaJapan misconfigured theirs). Proper canonical tags prevent duplicate content issues.

**Actions:**
```html
<link rel="canonical" href="https://akihabaratcg.com/pokemon/japanese/booster-boxes" />
```

#### 9. Build Strong Internal Linking
**Why:** 80% of competitors have 20+ internal links. PlazaJapan leads with 347 links. Internal links distribute page authority and help Google discover pages.

**Actions:**
- Add "Shop by Game" navigation with links to all game categories
- Add "Related Products" / "You May Also Like" sections
- Add "Popular Categories" section in footer
- Cross-link between Japanese and English versions of same product
- Add breadcrumb links on all pages

#### 10. Content Strategy: Add Category Descriptions
**Why:** Top competitors (Cardmarket, PlazaJapan) include descriptive text on category pages that helps with long-tail keywords.

**Actions:**
- Add 100-200 word descriptions on each category page
- Example Pokemon category: "Browse our authentic Pokemon TCG collection, sourced directly from Japan. We carry Japanese and English booster boxes, elite trainer boxes, booster bundles, and singles from the latest sets including..."
- Include trust signals: "All products are 100% authentic, sourced directly from Japanese distributors"

#### 11. Add Trust Signals
**Why:** Top competitors use "authentic", "secure", "official", "trusted" throughout their content. This improves both SEO and conversion.

**Actions:**
- "100% Authentic - Sourced directly from Japan"
- "Trusted by X+ customers worldwide"
- "Ships from Akihabara, Tokyo"
- Show any certifications, business registrations
- Add Trustpilot/review integration

### LOW PRIORITY

#### 12. Meta Keywords Tag
**Why:** Only Hobby-Genki still uses meta keywords, and Google ignores them. However, they may help with some smaller search engines.

**Action:** Add a modest meta keywords tag as supplementary: `<meta name="keywords" content="Japanese TCG cards, Pokemon cards, One Piece cards, Akihabara card shop, trading card games">`

#### 13. Robots Meta Directives
**Why:** Cardmarket properly uses `index,follow`. Some sites (PlazaJapan) accidentally use `noindex` which blocks them from search results.

**Action:** Ensure all public pages have `<meta name="robots" content="index, follow">` and NO pages accidentally have noindex.

#### 14. Sitemap and Robots.txt Optimization
**Why:** Essential for search engine crawling. Not visible from scraping but assumed.

**Actions:**
- Create comprehensive XML sitemap at `/sitemap.xml`
- Configure `robots.txt` to allow all important pages and block admin/api routes
- Submit sitemap to Google Search Console
